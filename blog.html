What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

Select by ID
document.getElementById('myDiv');

Select by class name
document.getElementsByClassName('myClass');

Select by tag name
document.getElementsByTagName('p');

Select by query selector
document.querySelector('p');

Select by query selector all
document.querySelectorAll('p');

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation in the DOM refers to the technique of attaching an event listener to a parent element instead of directly to each child element. This allows you to handle events for multiple child elements efficiently without having to add individual event listeners to each one.


Why is it useful?

Improved Performance: When working with dynamic content or large numbers of elements, event delegation can significantly improve performance. By attaching a single event listener to a parent element, you reduce the number of event listeners that need to be created and managed, leading to faster page load times and better responsiveness.
Simplified Code: Event delegation can make your code more concise and easier to maintain. Instead of writing separate event listeners for each child element, you can handle all events for a group of elements in a single place. This reduces code duplication and makes it easier to modify event handlers in the future.
Dynamic Content: Event delegation is particularly useful when dealing with dynamically generated content. You can attach an event listener to a parent element once, and it will automatically handle events for any child elements that are added to the DOM later.

How do you manipulate an element's attributes and styles using the DOM?

To access and modify an element's attributes using the DOM, you can use the following properties:

getAttribute(attributeName): Retrieves the value of the specified attribute.
setAttribute(attributeName, attributeValue): Sets the value of the specified attribute.
removeAttribute(attributeName): Removes the specified attribute from the element.
